Instrumentino

Instrumentino is an open-source modular graphical user interface framework for controlling Arduino based experimental instruments

Current project status:
  • Version 1: Currently available.
  • Version 2: Still in early development as of Nov. 2015. Any features, functionality, and documentation may change at any time. Contributers are welcome!

What is Instrumentino?

In a nutshell, Instrumentino aims to be a "control panel" to Arduino and similar devices in order to communicate with components like motors/pumps/sensors/lights/etc. Data collection from the components can be automated and controlling commands can be input either manually or automated. The overall goal is to enable you to quickly (and cheaply) build an "instrument" of one or more components, allowing you to get up and running with data collection/automation/etc. without having to spend time writing a lot of code.

Resources
